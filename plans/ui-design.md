# Sonet — UI/UX дизайн

## Философия дизайна

Название "Sonet" отсылает к сонету — поэзии, элегантности. UI должен ощущаться как запись в красивом кожаном блокноте: тёплый, тихий, сфокусированный. Никаких холодных корпоративных синих. Вместо этого — пергамент, янтарь, терракота, чернила.

---

## Типографика

| Использование | Шрифт | Fallback |
|--------------|-------|----------|
| Заголовки / Логотип | **Playfair Display** (serif) | Georgia, serif |
| Основной текст / UI | **Inter** (sans-serif) | system-ui, sans-serif |
| Код / Моноширинный | **JetBrains Mono** | Consolas, monospace |
| Контент редактора | **Inter** 16px / line-height 1.7 | — |

Playfair Display используется точечно: логотип, заголовок приложения, опционально заголовки заметок.

## Экраны приложения

### Общий макет (Layout)

```
┌──────────────────────────────────────────────────────────────┐
│ TopBar  [≡] [Breadcrumb: Workspace > Folder > Note] [🔍] [👤] │
├──────────────┬───────────────────────────────────────────────┤
│   Sidebar    │                                               │
│   240px      │              Main Content                     │
│              │                                               │
│ [Workspace   │                                               │
│  Switcher]   │                                               │
│              │                                               │
│ [+ Новая     │                                               │
│  заметка]    │                                               │
│              │                                               │
│ [Дерево      │                                               │
│  папок]      │                                               │
│   ▸ Папка    │                                               │
│     ▸ Под    │                                               │
│   ▸ Папка    │                                               │
│              │                                               │
│ [Теги]       │                                               │
│  # тег1      │                                               │
│  # тег2      │                                               │
│              │                                               │
│ [─────────]  │                                               │
│ [Граф]       │                                               │
│ [Настройки]  │                                               │
└──────────────┴───────────────────────────────────────────────┘
```

Sidebar сворачивается в 48px иконочную полосу на малых экранах или по нажатию.

---

### Экран 1: Вход / Регистрация

- Центрированная карточка на тёплом градиентном фоне (`#FBF8F4` → `#F3EDE4`)
- Логотип "Sonet" шрифтом Playfair Display сверху
- Подзаголовок: *"Ваши мысли, связанные воедино"*
- Поля: email + пароль, основная кнопка, ссылка переключения login/register
- Минимализм, без отвлечений

### Экран 2: Dashboard (Домашняя Workspace)

- **Приветствие**: "Добрый день, {имя}" с датой
- **Закреплённые заметки**: горизонтальный ряд карточек (заголовок, 2 строки, теги, дата)
- **Недавние заметки**: вертикальный список карточек
- **Быстрые действия**: кнопка "Новая заметка" (primary), "Новая папка" (secondary)

### Экран 3: Редактор заметки

- **Заголовок**: крупный текст (Playfair Display, 28px), без видимых границ
- **Мета-бар**: путь папки, теги (pills), кол-во слов, время последнего редактирования
- **Редактор**: TipTap rich-text, занимает остальное пространство
- **Bubble menu**: всплывающая панель форматирования при выделении текста (Bold, Italic, Strike, Code, Link, H1-H3, Lists, Quote)
- **Панель backlinks**: выдвижная правая панель с заметками, ссылающимися на текущую
- **`[[` автокомплит**: при вводе `[[` — выпадающий список заметок workspace

### Экран 4: Граф связей

- Полноэкранный canvas в main content
- **Ноды** = заметки (круги, размер по кол-ву связей, цвет `--color-accent-soft`)
- **Рёбра** = ссылки (кривые линии, цвет `--color-border`)
- Выбранная заметка выделена `--color-primary`
- Управление: зум, вписать всё, фильтр по тегу, поиск
- Клик на ноду → переход к заметке
- Реализация: `d3-force` → SVG

### Экран 5: Результаты поиска

- Строка поиска сверху (предзаполнена запросом)
- Фильтры: мульти-выбор тегов, выбор папки, диапазон дат
- Результаты: список карточек с подсветкой совпадений в заголовке и контенте

### Экран 6: Настройки Workspace

- Вкладки: **Общее** | **Участники** | **Теги**
- **Общее**: название, описание, иконка, опасная зона (удаление)
- **Участники**: список с аватарами, бейджи ролей, форма приглашения (email + роль), управление
- **Теги**: сетка тегов с цветами, кол-вом заметок, редактирование

### Экран 7: Профиль пользователя

- Загрузка аватара
- Display name, username (read-only), email
- Смена пароля
- Переключатель темы (light/dark) с live preview
- Кнопка выхода

## Ключевые UI-компоненты

### Sidebar (240px)
- Фон: `--color-bg-sidebar`
- Workspace switcher: dropdown с иконкой + названием
- Кнопка "Новая заметка": `--color-primary` фон, скруглённая 8px, высота 40px
- Дерево папок: иконки папок (outline), hover `--color-bg-hover`, active `--color-bg-active` + 3px левый бордер `--color-primary`

### Карточка заметки (NoteCard)
- Фон `--color-bg-surface`, бордер 1px `--color-border`, скругление 12px, padding 16px
- Заголовок: Inter 16px semibold
- Превью: 2 строки, `--color-text-secondary`, 14px
- Нижняя строка: теги (pills) слева, время справа (`--color-text-tertiary`, 12px)
- Hover: тень `0 2px 8px rgba(45,32,24,0.08)`

### Tag Pill
- Pill shape (rounded-full), padding 4px 10px, 12px
- Фон: цвет тега с 15% opacity
- Текст: цвет тега на полной насыщенности

### Bubble Menu (редактор)
- Всплывает над выделенным текстом
- Тёмная поверхность с белыми иконками
- Активная кнопка: `--color-primary`

### Модальное окно Share/Invite
- Overlay с backdrop blur
- Карточка 480px, скругление 16px, padding 24px
- Email + dropdown роли + кнопка "Пригласить"
- Список текущих участников с аватарами и ролями
